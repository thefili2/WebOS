## Features

### Desktop Environment

- XFCE-inspired desktop UI
- Custom wallpapers and themes
- Draggable desktop icons
- Grid snapping for icons
- Persistent desktop layout
- Taskbar with running applications
- Start menu with app search
- Multi-window support
- Window controls:
  - Drag
  - Resize
  - Minimize
  - Maximize


## Included Apps

### File Explorer
A browser-based file manager using a custom virtual filesystem.

### Terminal
A terminal-inspired interface for interacting with the system.

### Notepad
A simple text editor.

### Paint
A small drawing application.

### Web Browser
A built-in browser experience.

### Video Player
Play videos directly inside WebOS.

### Settings
Customize system preferences, themes, and options.

### Calculator
Supports chained calculations.

### Music Player
Features:
- Audio uploads
- Playback controls
- Seek bar
- Volume control

### Image Viewer
Features:
- Image viewing
- Zoom
- Fit-to-screen mode

### Task Manager
View running applications and active windows.
